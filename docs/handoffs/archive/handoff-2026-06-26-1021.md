## Cost basis (for $/100 questions)

From the pipeline-economics agent (`apps/quiz-pack-api`): generating **100 final questions through the generation model ≈ ~52K input + ~85K output tokens** (live v3 prompt `question_generation_v3_fact_first.md`, **best-of-N 3× over-generation** as currently configured → 300 raw generated, 10 kept per call × 10 calls; **~3× cheaper if over-gen disabled**). **Output dominates**, so cost tracks the **output $/M** most.

**Formula:** `$/100q ≈ 0.052 × ($/M in) + 0.085 × ($/M out)`.

⚠️ **Reasoning/thinking models** also emit hidden reasoning tokens billed at output rate → real cost > sticker, and creative work doesn't need heavy reasoning — prefer non-thinking or low effort.
