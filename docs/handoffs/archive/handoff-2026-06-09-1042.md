### How to get the data onto prod (step-by-step)

Source content lives in repo-root `chroma_data/chroma.sqlite3` (`quiz_questions` collection: 308 approved / 272 pending). Destination is the **`quiz-pack-db`** Fly Postgres (legacy flypg, image `quiz-pack-db:pgvector-0.8.2`, region cdg, started+healthy). The app's `DATABASE_URL` lives only as a Fly secret on `quiz-pack-api` and **cannot be read back**; the repo `.env` has a placeholder for `PROD_DATABASE_URL`.

**1. Open a tunnel to prod Postgres (leave running):**
```bash
fly proxy 15432:5432 -a quiz-pack-db
```

**2. Get the superuser password (legacy flypg env var on the DB machine):**
```bash
fly ssh console -a quiz-pack-db -C 'printenv OPERATOR_PASSWORD'
```
(User is `postgres`. If that var is empty, try `SU_PASSWORD`. The DB name is most
likely `quiz_pack` — same as local; confirm in step 3 if the connection fails.)

**3. Build the URL and DRY-RUN first (no writes):**
```bash
cd apps/quiz-pack-api
URL="postgresql://postgres:<PASSWORD>@localhost:15432/quiz_pack"   # adjust db name if needed
.venv/bin/python scripts/migrate_chroma_to_postgres.py \
    --chroma-path ../../chroma_data --database-url "$URL" --dry-run
```
Expect in the summary:
- `Skipped (not approved): 272`
- `Coerced true_false → text: 5`
- `Questions to migrate: 308`
- new-inserts vs upserts split tells you how much prod already had.

If connecting fails with an unknown-database error, list DBs to find the real name:
`psql "postgresql://postgres:<PASSWORD>@localhost:15432/postgres" -c "\l"`.

**4. EXECUTE (prod write — idempotent upsert on id, non-destructive):**
```bash
.venv/bin/python scripts/migrate_chroma_to_postgres.py \
    --chroma-path ../../chroma_data --database-url "$URL" --execute
```

**5. Verify the prod count (closes R15):**
```bash
psql "$URL" -c "SELECT review_status, count(*) FROM questions GROUP BY 1 ORDER BY 1;"
psql "$URL" -c "SELECT count(*) FROM questions WHERE review_status='approved' AND pack_id IS NULL;"
```
Target ≈ 308 approved. Then smoke-test the voice quiz against prod and confirm the 3
previously-empty categories now return questions: **wizarding-world (30), superheroes (34),
sports-mix (30)**.
