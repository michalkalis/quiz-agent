## Decisions

**`nonisolated final class` over per-method annotation:** `SWIFT_APPROACHABLE_CONCURRENCY=YES` makes the whole class `@MainActor`. Annotating each override individually with `nonisolated` is possible but fragile (any new override added without it will break). Class-level `nonisolated` is the clean solution; the test methods that need main actor are already explicitly `@MainActor`.

**TSAN split deferred:** Disabling TSAN at the scheme Test action level would affect HangsTests unit tests, which rely on race detection. A `.xctestplan` with per-bundle config is the right fix but is its own piece of work. Committed what's done (compile + scheme wire) rather than either reverting or doing a broader infra change unsolicited.
