## 1. Apple Foundation Models framework — unified Swift API for on-device + Claude + Gemini (WWDC 2026, June 9)

**Sources:**
- [What's new in the Foundation Models framework — WWDC26 — Apple Developer](https://developer.apple.com/videos/play/wwdc2026/241/)
- [WWDC 2026 Developer Tools: Foundation Models Now Swaps AI Providers Without Code Changes — TechTimes](https://www.techtimes.com/articles/318039/20260609/wwdc-2026-developer-tools-foundation-models-now-swaps-ai-providers-without-code-changes.htm)
- [Apple Expands Foundation Models at WWDC 2026 — TechResearchOnline](https://techresearchonline.com/news/apple-foundation-models-wwdc-2026-ai-framework-update/)

**Summary:** Apple announced a new `LanguageModel` protocol that lets one codebase target on-device AFM, Anthropic's Claude, or Google Gemini interchangeably — swapped via Swift Package Manager with no downstream code changes. Anthropic and Google publish official Swift packages. Private Cloud Compute (PCC) adds a 32K-context server model with configurable reasoning levels, no API keys, and no prompt storage. A new Dynamic Profiles API supports declarative multi-mode/multi-agent sessions. The framework goes open source this summer; iOS 27 / Xcode 27 target. Small-Business-Program developers (<2M App Store downloads) get PCC access at no cloud cost.

**Relevance to us:** Our iOS app today calls the backend for all LLM work. The `LanguageModel` protocol means lightweight tasks (answer validation preamble, hint generation, score display copy) could move on-device via PCC with zero API-key management in the app binary. The Anthropic Swift package lets us call `claude-sonnet-4-6` directly from Swift if needed. Dynamic Profiles maps naturally to our quiz state machine (listening → evaluating → hinting). **Watch only** — iOS 27 ships fall 2026; worth prototyping one on-device path now to be ready for launch.

## 2. GPT-5.6 expected late June — not released, nothing actionable yet

**Sources:**
- [GPT-5.6: OpenAI Chief Scientist calls it a meaningful leap — TechTimes](https://www.techtimes.com/articles/318492/20260616/gpt-56-openai-chief-scientist-calls-it-meaningful-leap-june-launch-nears.htm)
- [OpenAI Could Launch GPT-5.6 This Month — AndroidHeadlines](https://www.androidheadlines.com/2026/06/openai-gpt-5-6-release-date-chatgpt-overhaul-ipo-plans.html)

**Summary:** No official release as of today. Prediction markets (Polymarket/Metaculus) assign 83% odds to a June 22–28 window. Claimed features: 1.5M context window, improved token efficiency, stronger agentic reasoning. No model card, benchmark numbers, or API string published. Chief scientist Pachocki described it as "a meaningful leap over GPT-5.5."

**Relevance to us:** We use `gpt-4o` / `gpt-4o-mini` for generation and `gpt-4.1-mini` in the scoring A/B experiment. If GPT-5.6 prices competitively against `gpt-4o`, it could displace our generation model. **Watch only** — revisit when model card and pricing are published.
