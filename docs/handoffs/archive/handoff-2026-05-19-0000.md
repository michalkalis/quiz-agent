## Decisions

**pbxproj fix prístup — `TEST_TARGET_NAME` nie `BUNDLE_LOADER`:** HangsUITests je `com.apple.product-type.bundle.ui-testing` (nie unit test bundle). UI test bundle používa `TEST_TARGET_NAME = Hangs` (hovorí XCUITest kde hľadať app binary); unit test bundle používa `BUNDLE_LOADER` + `TEST_HOST`. Zámena by spôsobila iný typ crasha.

**`PRODUCT_NAME = "$(TARGET_NAME)"` namiesto literal `HangsUITests`:** Konzistentné s `HangsTests` konfiguráciami. Oba sú funkčne ekvivalentné ale `$(TARGET_NAME)` je idiomatickejšie.

**Záloha pbxproj pred editom:** Uložená do `/tmp/claude/project.pbxproj.bak` — len pre túto session, nie perzistentná.
