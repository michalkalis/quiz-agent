The committed `apps/quiz-pack-api/app/db/models/question.py:30` already imported `GenerationProvenance`, making the stash load-bearing. Lesson for future: before stashing, grep to see if any committed code already references the stashed symbols.
- **Rename over pytest_plugins workaround** — `pytest_plugins` pointing at the same module auto-discovered by conftest naming is a known pytest error. Renaming the module (removing `conftest` from filename) is cleaner than `--ignore` flags or conftest restructuring.
