CarQuiz is a hands-free trivia app. Driving is an important use case but NOT the only one — the app should work well hands-free in any context.

**Why:** The user wants broad appeal, not a niche driving-only app. Hands-free is a differentiator but the app should also be enjoyable when looking at the screen.

**How to apply:**
- Don't frame every feature decision around "driving safety" — frame around "hands-free first, visual second"
- Image questions are fine — another person nearby can answer, or user can glance
- Don't deprioritize visual features just because they require looking at the screen
- Multiplayer is planned post-MVP: voice calls with friends/family to play together or competitively
- Offline mode is NOT needed for now
