OpenAI's Chief Scientist described GPT-5.6 as a "meaningful improvement" over GPT-5.5, with a Polymarket consensus of ~83% probability for a June 22–28 launch. As of June 23 no official API model string exists. Rumoured (unverified) gains: context window pushed to 1.5M tokens (up from 1M in GPT-5.5), sharper long-horizon coding, and lower API pricing.

If GPT-5.6 ships with a confirmed lower per-token price at similar or better quality than GPT-5.5, it becomes a candidate for our question-generation calls. Do not plan any migration before the official API model ID lands; the rumoured specs are community extrapolation only. **Watch only — re-evaluate when the model string appears in the API.**
